## 📝 제출물 가이드

- 매주 커리큘럼에 맞는 MD 파일 제출
- 파일명: `이름.md`
- MD 내용:
  - 챕터 요약
  - 인상 깊었던 내용
  - 딥다이브 및 인사이트(Optional)

## 🔀 Git 컨벤션

### 브랜치 네이밍

```
{이름 영어 이니셜}
```

**예시:**

- `mjkim`

### 커밋 메시지

```
[partXX/이름] 커밋 내용
```

**예시:**

- `[part01/홍길동] 요약 및 정리 작성`
- `[part01/홍길동] 딥다이브 문서 추가`

## 🎯 스터디 진행 방식

- 스터디 전까지 주차별 요약 MD 커밋 업로드
- 스터디 시간마다 가장 인상 깊었던 인사이트 1개 공유
